# py-xiaozhi文档目录

本目录包含了 py-xiaozhi 项目 的全部功能文档，按功能模块进行划分，便于查阅和使用。

项目默认启用了以下模块：音乐、灯光、音量、相机 IoT 控制。
其他模块可根据需求自行扩展与启用。
视觉识别功能 需配置 智普大模型的 API Key 才可使用。
